# hive-mind

Hive Mind collective intelligence system for advanced swarm coordination.

## Usage

```bash
npx gendev hive-mind [subcommand] [options]
```

## Subcommands

- `init` - Initialize hive mind system
- `spawn` - Spawn hive mind swarm
- `status` - Show hive mind status
- `resume` - Resume paused session
- `stop` - Stop running session

## Examples

```bash
# Initialize hive mind
npx gendev hive-mind init

# Spawn swarm
npx gendev hive-mind spawn "Build microservices"

# Check status
npx gendev hive-mind status
```
