## Root Cause Analysis

1. **Transitive Type Dependencies**: Some installed packages (express, fastmcp, etc.) reference type definitions that are not explicitly installed
2. **TypeScript Configuration**: `skipLibCheck: true` should prevent this, but TypeScript is still attempting to resolve implicit type libraries
3. **Module Resolution**: Using `moduleResolution: "bundler"` may be causing TypeScript to expect all type definitions to be present
