## 🎯 Quick Start (NPX)

### Try It Now (No Installation)

```bash
# Multi-agent swarm (5 agents)

# Simple tasks (3 agents, faster)
```

## 🆕 New Features

### 1. Multi-Agent Swarm Decomposition

Every research task is automatically decomposed into specialized perspectives:

```
Your Task
    ↓
┌─────────────────────────────────────┐
│ 🔍 Explorer (20%)                   │
│ 🔬 Depth Analyst (30%)              │
│ ✅ Verifier (20%)                   │
│ 📈 Trend Analyst (15%)              │
│ 🧩 Synthesizer (15%)                │
└─────────────────────────────────────┘
```

### 2. Adaptive Swarm Sizing

- **Depth 1-3** (Simple): 3 agents
- **Depth 4-6** (Medium): 5 agents
- **Depth 7-10** (Complex): 7 agents

### 3. Priority-Based Execution

- **Priority 1**: Research agents (parallel)
- **Priority 2**: Verification agents (parallel)
- **Priority 3**: Synthesis agent (sequential)

### 4. New CLI Options

```bash
--single-agent           # v1.0.1 behavior (1 agent)
--swarm-size <number>    # Override adaptive sizing (3-7)
--max-concurrent <number> # Max parallel agents (default: 4)
--verbose                # Show all agent outputs
```
